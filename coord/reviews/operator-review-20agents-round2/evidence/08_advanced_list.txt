Engineering-tier verbs (hidden from default --help):

  burst                Temporarily route all traffic to one engine.
  coord                Coordinator commands: plan, run, integrate, status.
  engines-cooldowns    Show active engine cooldowns.
  engines-reliability  Show / publish engine reliability ranking from campaign data.
  l5-banner-demo       W11-L5-OUTPUT-CONTRACT: render a sample L5 ESCALATION banner.
  lint-spec            Pre-flight: validate a markdown spec for plan-readiness.
  lock                 Exclusive routing lock (disables auto-routing).
  panic-dump           Capture a secret-scrubbed snapshot of harness state into one tarball.
  proxy                Stateful 4-key API proxy with circuit breaker.
  replay               Reconstruct the dispatch (or v2 coord run) lifecycle for TASK_ID.
  spec-init            Scaffold a starter spec markdown with canonical sections.
  spec-register        Register a spec's SHA256 + author into the provenance log.
  spec-verify          Verify a spec's on-disk SHA matches its provenance registration.
  swarm-verify         Verify the last (or named) swarm actually wrote the expected paths.

Invoke via `harness <verb>` — hidden only affects help discovery, not callability.
